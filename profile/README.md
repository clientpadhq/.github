# ClientPad

ClientPad is open-source WhatsApp-first CRM infrastructure for service businesses.

It gives developers the SDK, API, CLI, dashboard, and WhatsApp automation layer to build lead capture, bookings, payments, follow-ups, reviews, and client workflows into any app.

## What ClientPad helps you build

- WhatsApp lead capture.
- Service booking flows.
- Paystack payment workflows.
- Follow-up reminders.
- Review requests.
- Client pipelines.
- Developer-friendly CRM APIs.
